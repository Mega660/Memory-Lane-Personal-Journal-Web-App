# Memory Lane – Personal Journal Web App

Memory Lane is a simple and elegant personal journal application that allows users to record, edit, and manage their daily memories directly in the browser.
All entries are saved using LocalStorage, meaning data persists even after refreshing the page.


##  Features
-  Add journal entries with title, date, and content
-  Edit existing entries
-  Delete entries with confirmation
-  Data persistence using LocalStorage
-  Clean and responsive UI
- Smooth fade-in animation for entries

## Built With
- HTML5
- CSS3 (Flexbox, Animations, Gradient Background)
- Vanilla JavaScript (DOM Manipulation, LocalStorage)
